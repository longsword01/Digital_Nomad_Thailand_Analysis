# Digital_Nomad_Thailand_Analysis
This project explores global digital nomad travel patterns, analyzing where nomads are based and which cities they visit most. Using Excel  and Tableau for visualization, the dashboard breaks down movement from a country level down to the most popular cities worldwide and within Thailand
## Tools Used
- Excel — data cleaning and transformation
- Tableau — dashboard and visualizations
## Key Insights
- Top countries where digital nomads are based
- Most visited cities globally
- Most popular destinations within Thailand
